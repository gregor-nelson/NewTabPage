# New Tab Dashboard

A minimal new tab page with a digital clock, calendar, and date display.

## Features

- Digital clock with 12/24-hour format
- Full date line with ISO week numbers
- Monthly calendar with navigation
- Customizable settings (saved locally)
- Dark mode design
- No tracking, no ads

## Usage

Click the gear icon (top right) to customize:
- Time format (12/24-hour)
- Show/hide seconds
- Week start day (Monday/Sunday)
- Show/hide week numbers

Keyboard shortcuts:
- `ESC` - Close settings

Settings sync across devices if you're signed into Chrome.

## Privacy

This extension collects no data. Settings are stored locally on your device and never leave your browser. See [PRIVACY.md](PRIVACY.md) for details.
